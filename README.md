# This is your typical jscode program

# Install js in linux (ubuntu\debian versions) (node)

## First of all you gotta update

```bash
$ sudo apt update
# or
$ sudo apt-get update
```

## Install node

```bash
$ sudo apt install nodejs
```

# Check if node is installed

## Run node version

```bash
$ node -v
```

## This will print the version For example,

```
v10.19.0
```

# Compiling

```
$ node main.js
```
